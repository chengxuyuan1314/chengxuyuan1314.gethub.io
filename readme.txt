  1.本次是写了一个手机购的网站，主要是采用响应式布局，利用bootstrap 和 媒体查询进行完成，网站主题是一个购物平台。兼容大部分设备但对width为360px以下设备没有进行兼容。
  2. 网站共6个页面，首页 index.html，推荐页 recommend.html,详情页detail.html,
登录页login.html,注册页register.html,排名页ranking1.html。
  3.目录文件有bootstrap，css，script，images（放置背景图等布局图），upload（放置产品图）。
  4.css放有专有样式 公有样式 和基础样式 由于本人对javascript没有学习， script就写了3个  两个轮播图和一个简单的表单验证。
  5.主页产品区使用了webp格式图片，可能对部分游览器不兼容。
  6.对网站加了icon 以及SEO优化。
  7.主页主要是设计了一个轮播图，一个新品推荐模块，小米模块，京东模块.
  8.推荐页主要是一个旋转动画推荐的是华为mate3pro.
  9.详情页也就是一个轮播图和产品参数。
  10.登录和注册页公用头部 主要内容是表单布局。
  11.排名是对现在手机的性能进行排名 ，主要利用了flex布局。
